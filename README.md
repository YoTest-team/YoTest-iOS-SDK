#### <a name="Compile">编译(自定义)</a>

若有自定义更改 SDK 的需求，可将仓库克隆到本地，自行修改后，执行编译脚本打包 SDK

```
git clone https://github.com/YoTest-team/YoTest-iOS-SDK.git
cd YoTest-iOS-SDK
./build.sh
```
